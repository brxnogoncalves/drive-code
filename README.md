# 🚗👨‍💻 Drive <Code/>

---

### **📚 Sobre o projeto**

O Drive & Code é um website informativo que conta a história de como meu gosto por carros me levou a aprender tecnologia na prática.

O projeto apresenta experiências reais com desenvolvimento de soluções automotivas, explorando como sistemas eletrônicos e softwares estão ligados ao funcionamento, desempenho e conforto dos veículos.

Além de ser informativo, o website permite que usuários se cadastrem e compartilhem suas preferências, gerando dados que podem ser vizualizados em uma dashboard.

### 🐱‍🏍 Como executar o projeto

1. Clone o repositório:
   `git clone https://github.com/brxnogoncalves/drive-code`
2. Instale as dependências:
   `npm install` || `npm i`
3. Configure o banco de dados MySQL
4. Inicie o servidor:
   `npm start`
5. Acesse no navegador:
   `http://localhost:3000`
